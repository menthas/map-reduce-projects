
The script assumes that ssh to localhost is done, and hadoop NameNode daemon and ResourceManager daemon are up and running. Also the input file is assumed to be kept on namenode.

Example Run:

./run.sh /home/sahil/workspace/Assignment3/data/data.csv /home/sahil/workspace/Assignment3/data/output_directory /home/sahil/workspace/Assignment3/data/predict.csv /home/sahil/workspace/Assignment3/data/check.csv
